# Contributing

Thank you for helping Rodavarion TDriver.

1. Describe bugs with the Linux distribution, desktop session (X11/Wayland), application version, device, and reproduction steps.
2. Keep changes focused and avoid unrelated formatting rewrites.
3. Never include passwords, private keys, access tokens, personal logs, or device serial numbers in reports.
4. Run `./tools/preflight_arch.sh --edition full` before proposing a change.
5. Contributions are accepted under the MIT License.

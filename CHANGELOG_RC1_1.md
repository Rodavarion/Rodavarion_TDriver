# Rodavarion TDriver 1.0.0 RC1.2 — Browser fallback fix

- Added an explicit web-browser application profile.
- Added a configurable default profile for all unmatched graphical applications.
- Preserved terminal-specific Ctrl+Shift+C / Ctrl+Shift+V behavior.
- Prevented unmatched browsers from being treated as an unknown/unsafe context.
- Added regression tests for Firefox, Chromium, Konsole, and unknown applications.

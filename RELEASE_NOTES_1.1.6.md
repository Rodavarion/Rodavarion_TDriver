# Rodavarion TDriver 1.1.6

- Live monitor now exclusively captures and safely relays events while the daemon is paused.
- System tray self-heals after Plasma/KWin restart and suspend/resume.
- Per-device Configure buttons added; double-click remains supported.
- Generic HID coverage expanded for mice, ROG Gladius III Wireless AimPoint, gamepads, HOTAS, wheels, pedals, remotes, presenters, arcade and 3D controllers.
- Unexpected 200% monitor scaling guard using KScreen profiles.
- No desktop shortcut is created.
- Full uninstall removes Rodavarion TDriver data by default.
- Build-only orphan dependencies installed by TDriver are cleaned after installation.

# Безпека системного налаштування

- Немає тихого встановлення пакетів.
- Немає збереження паролів.
- Використовуються лише офіційні репозиторії Arch Linux.
- AUR не використовується автоматично.
- Назви пакетів і модулів беруться з allow-listed маніфесту.
- Системні служби вмикаються лише після окремого підтвердження.
- Програма не запускається від root.
- Primary pointer interface не захоплюється.
- Firmware/HID write path залишається вимкненим.

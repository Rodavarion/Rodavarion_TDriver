# TDriver Runtime Critical Fixes

Виправлено:

1. `DeviceControl::Unknown` більше не перетворюється на ліву кнопку.
2. `ActionType::Disabled` тепер вимагає ексклюзивного перехоплення бічної кнопки.
3. `ActionEngine` кешує профіль призначень і перечитує JSON лише після зміни файлу, його розміру або ключа пристрою.
4. Повідомлення для `Default` і `Disabled` розділено для точнішої діагностики.
5. Додано конструктор `ActionEngine` із явним шляхом до mappings-файлу для тестованості.

Примітка: повну збірку слід перевірити на Arch Linux через штатний `tools/build_arch.sh`, оскільки середовище підготовки патча не містило Qt6 SDK.
